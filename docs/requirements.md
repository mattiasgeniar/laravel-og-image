---

This package requires **PHP 8.4+** and **Laravel 11+**.

Under the hood, this package uses [spatie/laravel-screenshot](https://github.com/spatie/laravel-screenshot) to generate the screenshots. By default, laravel-screenshot uses [Browsershot](https://spatie.be/docs/browsershot), which requires Node.js and a Chrome/Chromium binary.

You can find the Browsershot requirements [in its documentation](https://spatie.be/docs/browsershot/v4/requirements).
